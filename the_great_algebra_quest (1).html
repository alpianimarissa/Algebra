Mission 2: Algebraic Equilibrium

Two-Step Balance Scales

If you have an equation like $2x + 3 = 11$, how do we find the value of a single mystery box $x$? We must perform two fair steps to keep the scale balanced:

Step 1: Remove the extra weight (the constant $+3$) by subtracting $3$ from both sides of the balance scale!

$2x = 11 - 3 \implies 2x = 8$

Step 2: Divide the remaining weight into two equal groups!

$x = \frac{8}{2} \implies x = 4$

Mission 3: Algebraic Secret Language

Translating Sentences into Algebra

Mathematics is a cool universal language. When reading word puzzles or riddles, we can translate them directly into clean algebraic expressions using these handy keywords:

🔑 Algebraic Keyword Detector

  • "More than..." / "Added by" $+$ (Addition)
  • "Less than..." / "Difference of" $-$ (Subtraction)
  • "Times..." / "Each" $\times$ (Multiplication)
  • "Shared equally..." / "Split into" $/$ (Division)
